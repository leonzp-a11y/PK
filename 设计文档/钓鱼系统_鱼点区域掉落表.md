# 钓鱼系统_鱼点区域掉落表

更新时间：2026-05-13

## 1. 设计目的

本表用于定义“不同地图区域的鱼点可以钓出什么”。目标是让钓鱼掉落与地图环境、区域主题、探索进度形成关联，而不是所有水域共用同一套掉落池。

设计参考《魔兽世界》式区域钓鱼体验：

- 不同地图有自己的主要鱼类。
- 特殊水域有更高概率产出特定鱼或稀有物。
- 鱼点可以产出鱼，也可以产出杂物、旧物、区域线索。
- 稀有产出和区域内容绑定，让玩家愿意回到不同地图补收集。

## 2. 掉落表使用原则

| 原则 | 说明 |
|---|---|
| 区域绑定 | 每个地图区域至少有 2-3 种代表鱼类 |
| 鱼点差异 | 普通鱼点产基础鱼，特殊鱼点提高稀有或旧物概率 |
| 非鱼结果 | 少量杂物、旧物、漂流瓶等增强惊喜 |
| 不强迫刷 | 稀有鱼偏收集与轻奖励，不绑定核心战力 |
| 可扩展 | 后续可加入天气、时间、鱼饵、任务进度条件 |

## 4. 鱼类分层建议

| 分层 | 说明 | 使用方式 |
|---|---|---|
| 普通鱼 | 大多数鱼点可获得 | 出售、食材、基础图鉴 |
| 区域鱼 | 只在特定区域出现 | 区域收集目标 |
| 稀有鱼 | 低概率出现，部分绑定特殊鱼点 | 图鉴、成就、兑换 |
| 传说鱼 | 每个大区域最多 1 条 | 后续版本或活动目标 |

第一版不建议把鱼类数量铺得太大。每个区域 2 种普通鱼 + 1 种稀有鱼即可成立。

## 5. 非鱼掉落分层建议

| 分层 | 说明 | 示例用途 |
|---|---|---|
| 普通杂物 | 轻度幽默或少量兑换 | 破靴子、湿木枝 |
| 区域旧物 | 和地图主题有关 | 矿工纽扣、商队铜牌 |
| 线索物 | 可引出后续内容 | 漂流瓶、旧信、航海残页 |
| 委托物 | 可交给 NPC | 贵族戒指、旧军牌 |

非鱼掉落不宜太高频。它的价值在于“偶然钓到不该在水里的东西”。

## 6. 第一版建议落地范围

第一版可以先做 3 个区域：

| 区域 | 原因 |
|---|---|
| 晨露溪谷 | 新手教学，普通鱼点与发光鱼点 |
| 黑松森林 | 森林主题，适合做隐藏旧物 |
| 旧矿坑地下河 | 与已有矿洞探索概念衔接，适合做区域线索 |

第一版规模：

- 3 个区域。
- 6 个鱼点。
- 8-10 种鱼。
- 3 种稀有鱼。
- 6-8 个非鱼掉落。
- 每个区域 1 个可记忆的特殊掉落。

## 7. 后续扩展方向

后续可逐步加入：

- 鱼饵影响掉落权重。
- 天气、昼夜影响特殊鱼。
- 地图探索度解锁特殊鱼点。
- NPC 委托指定鱼类或旧物。
- 区域鱼王或限时钓鱼活动。

这些内容建议放在 P2/P3 阶段，不放入第一版强制范围。
